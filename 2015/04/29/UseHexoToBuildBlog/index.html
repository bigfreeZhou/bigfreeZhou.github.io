使用hexo和GitCafe搭建独立博客

发表于 2015-04-29   |   分类于 other   |  

准备工作

本教程采用 hexo + GitCafe Pages + Markdown实现独立博客构架。

hexo

hexo是一个基于node.js的静态博客生成程序，支持Markdown语法，可以快速生成静态网页。

GitCafe Pages

GitCafe是GitHub的山寨国产版本，由于众所周知的原因速度比GitHub快上不少，并且提供类似与GitHub Pages的静态网页托管服务GitCafe Pages.

Markdown

Markdown 是一种简单的标记语言，相比HTML，Markdown语法更为简单，阅读和书写更为方便。这是一个简单的Markdown教程。

hexo搭建本地博客

安装node.js

hexo命令

  • 初始化： hexo init
  • 添加新文章： hexo n
  • 生成静态页面： hexo g
  • 本地运行： hexo s
  • 部署： hexo d

建立本地博客

  • 创建博客目录
  • 在cmd环境下进入博客目录，执行hexo init初始化
  • 执行hexo g生成静态页面
  • 执行hexo s
  • 浏览器中访问http://localhost:4000/，页面粗线了~

发表新文章

新文章保存在博客目录\source\_posts下，使用Markdown语法书写，保存为.md文件，Markdown可以采用普通编辑器编辑，也可以采用支持Markdown语法高亮的高级编辑器，我使用的是SublimeText。

再次运行hexo g和hexo s就可以看到新文章啦。

使用主题

hexo的默认主题还不错，如果不喜欢，hexo还提供了不少第三方主题供你选择，新世界的大门。
我选用了Jacman主题，主题使用的方法可以参考这里。
下载主题需要使用Git，没有接触过可以参考此教程Pro Git。
